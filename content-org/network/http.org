* HTTP 1.1

缺点：

队首阻塞、浏览器并发请求数限制

即使是为了请求一个非常少的数据，也可能需要完整经历：建立 TCP 连接 => 发送 HTTP 请求 => 服务端处理 => 返回响应数据整个过程

** 状态码

- 301

 Moved Permanently

 代表资源的永久重定向

- 302

 Found

 代表资源的临时重定向

- 304

  Not Modified

** 请求方法

- Post

  新建资源

- Put

  更新资源
