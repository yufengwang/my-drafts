## Character Class {#character-class}

字符类

| 字符 | 含义                             |
|----|--------------------------------|
| `\d` | 数字                             |
| `\D` | 非数字                           |
| `\s` | 空格，换行符，制表符             |
| `\S` | 非 \s                            |
| `\w` | 拉丁字母，数字，下划线 _         |
| `\W` | 非 \w                            |
| `.`  | 除 \n 外的任意字符; 如果有 `s` flag，则为任意字符 |


## Quantifiers {#quantifiers}

量词，用于针对其前面的字符的数量的修饰

| 字符  | 含义                    |
|-----|-----------------------|
| `{n}` | n个                     |
| `+`   | 1或多，[1, +\\(\infty\\) ) |
| `*`   | 0或多，[0, +\\(\infty\\) ) |
| `?`   | 0或1，{0, 1}            |
